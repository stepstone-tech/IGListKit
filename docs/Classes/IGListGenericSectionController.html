IGSTListGenericSectionController


@interface IGSTListGenericSectionController <__covariant ObjectType> : IGSTListSectionController

This class adds a helper layer to IGSTListSectionController to automatically store a generic object in didUpdateToObject:.

  • object

    The object mapped to this section controller. Matches the object provided in [IGSTListAdapterDataSource listAdapter:sectionControllerForObject:] when this section controller was created and returned.

    Note

    This object is briefly nil between initialization and the first call to didUpdateToObject:. After that, it is safe to assume that this is non-nil.

    Declaration

    Objective-C

    @property (readonly, strong, nonatomic, nullable) ObjectType object;

    Swift

  • -didUpdateToObject:

    Updates the section controller to a new object.

    Note

    This IGSTListSectionController subclass sets its object in this method, so any overrides must call super.
